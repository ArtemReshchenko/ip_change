# 🔐 IP Changer — Зміна IP при авторизації в кабінеті ДПС

## Що це робить

Коли ви обираєте ключ зі списку — програма автоматично змінює IP-адресу через NordVPN і відкриває сайт `cabinet.tax.gov.ua` для авторизації.

---

## Як запустити

### Крок 1: Скопіюйте папку на Mac

Скопіюйте всю папку `ip_change` на робочий стіл MacBook.

### Крок 2: Дозвольте запуск (тільки перший раз)

1. Відкрийте **Terminal** (Finder → Програми → Утиліти → Terminal)
2. Напишіть цю команду і натисніть Enter:

```
chmod +x ~/Desktop/ip_change/start.command
```

### Крок 3: Запустіть

Двічі клікніть на файл **`start.command`** у папці `ip_change`.

> **Примітка:** Якщо macOS попереджає про невідомого розробника — натисніть правою кнопкою → Відкрити → Відкрити.
> 
> 🛑 **Якщо пише "Файл пошкоджено" (Damaged):** Це захист macOS (Gatekeeper) для завантажених архівів. Відкрийте Terminal і виконайте команду: `xattr -rc ~/Desktop/ip_change` (натисніть Enter), після чого файл запуститься.

## Як користуватись

Після запуску ви побачите список ключів з прогресом:

```
  🔐 IP Changer — cabinet.tax.gov.ua

  Прогрес: ████████░░░░░░░░░░░░ 42/200 (21%)

  #     Ст   Ключ                          Країна VPN
  ──────────────────────────────────────────────────────
  1     ✓    ФОП Тестовий 1                United States
  2     ✓    ФОП Тестовий 2                Germany
  3     ○    ФОП Тестовий 3                France

  Введіть номер ключа | 'h' довідка | 'q' вихід

  ▸
```

**✓** = ключ вже оброблений, **○** = ще не оброблений

### Основні дії

| Що ввести | Що станеться |
|-----------|-------------|
| **Номер** (наприклад `3`) | Обрати ключ → змінити IP → відкрити сайт |
| **q** | Вийти з програми |

### Зручні команди

| Що ввести | Що станеться |
|-----------|-------------|
| **/ текст** | Пошук ключа по назві (наприклад `/Іванов`) |
| **//** | Скинути фільтр пошуку |
| **>** | Наступна сторінка (по 20 ключів) |
| **<** | Попередня сторінка |
| **p** | Показати тільки необроблені ключі |
| **h** | Показати довідку |
| **clear** | Скинути весь прогрес |

## Що станеться після вибору ключа

1. VPN відключиться від поточної країни
2. VPN підключиться до нової країни (якщо не вдасться — спробує ще 3 рази)
3. Програма перевірить що IP дійсно змінився
4. Якщо такий IP вже був у іншого ключа — покаже попередження ⚠
5. **Програма скопіює повний шлях до файлу ключа у ваш буфер обміну 📋**
6. Відкриється браузер з `cabinet.tax.gov.ua`
7. На сайті натисніть кнопку вибору файлу, і в діалоговому вікні просто натисніть **Cmd+V** (Вставити) → Enter.
8. Після успішної авторизації повертаєтесь у програму → Enter → обираєте наступний ключ

## Файли в папці

| Файл | Для чого |
|------|----------|
| `start.command` | 👈 **Запуск програми** (двічі клікніть) |
| `config.json` | Список ключів та їх країн VPN |
| `progress.json` | Автоматично зберігає прогрес (створюється сам) |
| `ip_changer.py` | Код програми (не чіпати) |
| `generate_config.py` | Генерація config.json з папки ключів (для адміна) |

---

## Як додати свої ключі

### Автоматично (рекомендовано)

```
python3 generate_config.py /шлях/до/папки/з/ключами
```

### Вручну

Відкрийте `config.json` в TextEdit:

```json
{
  "browser_url": "https://cabinet.tax.gov.ua/",
  "open_browser": true,
  "mappings": [
    { "key_file": "key_001.dat", "label": "Назва ключа 1", "country": "Germany" },
    { "key_file": "key_002.dat", "label": "Назва ключа 2", "country": "France" }
  ]
}
```
